Neler yaptım?

HTML, CSS, Boostrap, JavaScipt ve React ile web site arayüzleri ve hyper casual oyunlar yaptım. Yeni başladığım Wordpress ile de ara yüz tasarımlarının hem kodlama hem de hız anlamında bir adım öteye geçmek için kullanmaya başladım. Daha önceleri flutter ile geliştirdiğim mobil dünyasına bundan sonra React Native ile de mobil uygulama geliştirmeye devam edeceğim. Projelerim ve yaptığım işlerin bir kısmı Github ve Linkedin sayfamda mevcuttur. Ayrıca yakın zamanda Youtube üzerinden de oyun ve web sayfalarımın anlatımı ve yapılışı ile ilgili videolar yayınlamayı düşünüyorum.

Geçen yıllarda flutter ile mobil uygulamalar geliştirmiş fakat sektörde Native olarak daha fazla ihtiyaç olduması ve web ile mobili aynı anda yapmak maksadıyla React ve React Native ile devam etmeye karar verdim.
